## Goenv(Linux only)

Download corresponding package from office site then run follows as: 
```Shell
$ sudo tar -C /usr/local -xzf go$VERSION.$OS-$ARCH.tar.gz
$ export PATH=$PATH:/usr/local/go/bin` and add this line in your /etc/profile.

```
